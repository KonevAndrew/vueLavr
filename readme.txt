дерективы

v-show      имеет значение true или false, для отображения контента(d-none)

v-model     род.элемент => добавляет в него данные по значению дерективы

v-on:click  слушатель событий с событием клик, есть еще модификатры, once событие сработает 1 раз

v-bind      позволяет записать динмическое значение в html атрибут
